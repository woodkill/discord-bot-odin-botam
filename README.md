## 사용방법

모든 명령어는 . 으로 시작합니다.\

### 길드관련 명령어


#### .길드등록
다른 모든 기능을 사용하려면 먼저 당신의 디스코드 서버를 어떤 길드에서 사용하고 있는 지 "길드등록" 명령어를 이용해 등록해 주어야 합니다.
```
.길드등록 오딘서버명 길드명
```
#### .길드확인
길드확인 명령어를 이용하면 현재 등록되어 있는 길드를 확인할 수 있습니다.
```commandline
.길드확인
```
